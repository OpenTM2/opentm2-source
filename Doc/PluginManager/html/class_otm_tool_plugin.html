Stops the plugin. Terminating-function for the plugin, will be called directly before the DLL containing the plugin will be unloaded.
The method should call PluginManager::deregisterPlugin() to tell the PluginManager that the plugin is not active anymore. Warning: THIS METHOD SHOULD BE CALLED BY THE PLUGINMANAGER ONLY!
