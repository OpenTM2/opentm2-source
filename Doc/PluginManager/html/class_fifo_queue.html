Detailed Description

FifoQueue.H - Include file for the FifoQueue class Copyright (c) 1999-2012, International Business Machines Corporation and others. All rights reserved. Description: This file contains the class definition for the JSONize helper class.

class for a primitive FirstIn-FirstOut queue

This class is a singleton and provides functions for the processing of JSON formatted strings
