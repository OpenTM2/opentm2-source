int EqfSharedOnLanMemory::searchProposal ( OtmProposal &  SearchKey,
std::vector< OtmProposal * > &  FoundProposals,
unsigned long  ulOptions 
) [virtual]

Get a list of memory proposals matching the given search key.

This method uses the search data contained in the search key to find one or more matching proposals in the memory. At least the szSource and the szTargetLang members of the search key have to be filled by the caller. The caller provides a list of OtmProposals which will be filled with the data of the matching proposals. The number of requested proposals is determined by the number of proposals in the list.

Parameters:
SearchKeyproposal containing search string and meta data
FoundProposalsrefernce to vector with OtmProposal objects
ulOptionsoptions for the lookup
Returns:
0 or error code in case of errors
