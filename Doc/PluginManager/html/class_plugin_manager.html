Detailed Description

The PluginManager controls loading of the plugins and calling their appropriate member-functions. Plugins must register with the PluginManager.

The OpenTM2 PluginManager class is a singleton and represents only an interface.
To access the PluginManager object, use the static method getInstance(). An instance of this class will be created automatically.
Implementation details can be found in the PluginManagerImpl class.
