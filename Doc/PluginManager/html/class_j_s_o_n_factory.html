Detailed Description

JSON.H - Include file for the JSON factory class Copyright (c) 1999-2012, International Business Machines Corporation and others. All rights reserved. Description: This file contains the class definition for the JSONize helper class.

factory class for JSON related functions

This class is a singleton and provides functions for the processing of JSON formatted strings
